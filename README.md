# 任务管理系统 - 微信小程序

一个功能完整、设计优雅的微信小程序任务管理系统，采用像素风格 UI，支持时间规划、任务管理、日历规划等功能。

![微信小程序](https://img.shields.io/badge/微信小程序-07C160?style=flat-square&logo=wechat&logoColor=white)
![License](https://img.shields.io/badge/license-MIT-blue?style=flat-square)

## ✨ 功能特性

### 🕐 时间规划表
- **双计时器系统**：独立的工作时间和学习时间计时器
- **实时计时**：支持开始/停止计时，自动累计时间
- **时间规划**：添加和管理每日时间规划，记录工作/学习时间段
- **数据持久化**：计时状态自动保存，关闭小程序后仍可恢复

### 📋 今日任务栏
- **项目管理**：创建、删除项目，支持项目状态管理
- **任务管理**：为项目添加任务，标记完成状态
- **任务优先级**：支持不同优先级的任务显示
- **快速操作**：一键添加任务，快速删除项目

### 📁 三级任务管理
- **层级结构**：支持一级、二级、三级任务层级管理
- **SOP 步骤**：三级任务可作为 SOP（标准作业程序）步骤
- **任务完成**：支持任务完成状态切换
- **快速添加**：三级任务可快速添加到今日任务栏
- **灵活编辑**：支持编辑和删除各级任务

### 📅 日历规划
- **日历视图**：直观的日历展示，支持月份切换
- **日期选择**：点击日期可进行相关操作
- **事件管理**：支持在日历上添加和管理事件

## 🎨 设计特色

- **像素风格 UI**：采用复古像素风格设计，界面简洁美观
- **紫色粉色配色**：柔和的紫色粉色配色方案，视觉效果舒适
- **响应式布局**：完美适配手机屏幕，无需滚动即可查看所有内容
- **交互友好**：按钮大小合理，操作流畅自然

## 🛠️ 技术栈

- **框架**：微信小程序原生框架
- **语言**：JavaScript (ES6+)
- **存储**：微信小程序本地存储 (wx.setStorageSync)
- **架构**：模块化设计，服务层与页面层分离

## 🚀 快速开始

### 环境要求

- 微信开发者工具（最新版本）
- 微信账号（用于登录开发者工具）

### 安装步骤

1. **下载项目**
   ```bash
   git clone <repository-url>
   cd wen/miniprogram
   ```

2. **打开微信开发者工具**
   - 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 安装并登录微信开发者工具

3. **导入项目**
   - 打开微信开发者工具
   - 点击"导入项目"
   - 选择 `miniprogram` 文件夹作为项目目录
   - AppID 选择"测试号"（开发阶段）或填写你的小程序 AppID

4. **编译运行**
   - 点击工具栏的"编译"按钮（或按 `Ctrl+B`）
   - 在模拟器中查看效果

### 配置说明

- **AppID**：在 `project.config.json` 中配置，开发阶段可使用测试号
- **项目目录**：确保选择的是 `miniprogram` 文件夹，不是外层目录

## 📖 使用说明

### 时间规划表

1. **开始计时**
   - 点击"开始"按钮开始计时
   - 计时过程中按钮会禁用，防止重复点击
   - 时间实时更新显示

2. **停止计时**
   - 点击"结束"按钮停止计时
   - 时间会自动累加到总时间中
   - 显示本次计时时长

3. **添加时间规划**
   - 点击右上角"添加"按钮
   - 选择类型（工作/学习）
   - 填写标题和时间段
   - 确认添加

### 今日任务栏

1. **添加项目**
   - 点击右上角"添加"按钮
   - 输入项目名称
   - 项目默认状态为"进行中"

2. **添加任务**
   - 在项目下方点击"+ 添加"按钮
   - 输入任务名称
   - 任务会自动添加到项目中

3. **完成任务**
   - 勾选任务前的复选框
   - 已完成的任务会显示删除线

4. **删除项目**
   - 点击项目右下角的"删除"按钮
   - 确认删除操作

### 三级任务管理

1. **添加一级任务**
   - 点击右上角"+ 添加一级"按钮
   - 输入一级任务名称

2. **添加二级任务**
   - 在一级任务下点击"+ 添加二级"按钮
   - 输入二级任务名称

3. **添加 SOP 步骤**
   - 在二级任务下点击"+ 添加步骤"按钮
   - 输入步骤名称

4. **完成任务**
   - 勾选步骤前的复选框
   - 已完成步骤会显示删除线

5. **快速添加到今日任务**
   - 点击步骤右侧的箭头按钮
   - 步骤会自动添加到今日任务栏

### 日历规划

1. **切换月份**
   - 点击左右箭头按钮切换月份
   - 当前月份会显示在中间

2. **选择日期**
   - 点击日历上的日期
   - 可以进行相关操作（功能可扩展）

## 💾 数据存储

- **存储方式**：所有数据存储在微信小程序本地存储中
- **存储内容**：
  - 计时器状态（运行状态、累计时间）
  - 时间规划记录
  - 项目任务数据
  - 三级任务数据
  - 日历事件

**注意事项**：
- 清除小程序数据会丢失所有记录
- 数据不会在多设备间同步
- 建议定期备份重要数据

## 🏗️ 架构设计

### 模块化设计

项目采用模块化架构，代码结构清晰：

- **服务层（services）**：负责业务逻辑处理
  - 每个服务独立管理自己的数据
  - 提供统一的 API 接口
  - 数据持久化封装

- **工具层（utils）**：提供通用功能
  - 时间处理工具
  - 本地存储工具
  - ID 生成工具

- **页面层（pages）**：负责 UI 展示和交互
  - 通过服务层访问数据
  - 处理用户交互事件
  - 更新页面状态

### 代码特点

- ✅ **解耦设计**：服务之间相互独立，便于维护
- ✅ **可复用性**：工具函数可在多处使用
- ✅ **可扩展性**：易于添加新功能
- ✅ **代码规范**：统一的命名规范和代码风格

## 🔧 开发规范

- 使用 ES6+ 语法
- 模块化导入导出（import/export）
- 统一的命名规范（驼峰命名）
- 清晰的代码注释
- 像素风格 UI 设计

## 📝 注意事项

1. **数据安全**
   - 所有数据存储在本地，清除小程序数据会丢失
   - 建议定期备份重要数据

2. **计时器状态**
   - 计时器在页面关闭后仍会继续运行（状态已保存）
   - 重新打开小程序时会恢复计时状态

3. **兼容性**
   - 需要微信小程序基础库 2.2.1 及以上版本
   - 支持 ES6 模块导入导出

4. **性能优化**
   - 使用本地存储，读写速度快
   - 页面加载时一次性加载所有数据
   - 计时器更新频率为 1 秒
