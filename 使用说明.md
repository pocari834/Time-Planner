# 使用说明

## 快速开始

1. 在微信开发者工具中打开项目
2. 修改 `project.config.json` 中的 `appid` 为你的小程序 AppID（如果没有，可以先使用测试号）
3. 点击"编译"按钮即可运行

## 功能使用

### 1. 时间规划表

- **开始计时**：点击"开始"按钮开始计时，计时过程中按钮会禁用
- **停止计时**：点击"结束"按钮停止计时，时间会自动累加
- **添加时间规划**：点击右上角"添加"按钮，填写时间规划信息

### 2. 今日任务栏

- **添加项目**：点击右上角"添加"按钮，输入项目名称
- **添加任务**：在项目下方点击"添加任务"，输入任务名称
- **完成任务**：勾选任务前的复选框即可标记为完成
- **删除项目**：点击项目右下角的"删除"按钮

### 3. 三级任务管理

- **添加一级任务**：点击右上角"添加一级"按钮
- **添加二级任务**：在一级任务下点击"添加二级项目"
- **添加SOP步骤**：在二级任务下点击"添加SOP步骤"
- **完成任务**：勾选步骤前的复选框
- **添加到今日任务**：点击步骤右侧的箭头按钮，快速添加到今日任务栏
- **编辑/删除**：点击对应的编辑或删除图标

### 4. 日历规划

- **切换月份**：点击左右箭头切换月份
- **查看日期**：点击日期可以查看该日期的事件（功能待扩展）

## 数据存储

所有数据存储在微信小程序的本地存储中，包括：
- 计时器状态
- 时间规划记录
- 项目任务数据
- 三级任务数据
- 日历事件

**注意**：清除小程序数据会丢失所有记录，请谨慎操作。

## 技术说明

- 使用 ES6+ 语法
- 模块化设计，代码解耦
- 服务层管理业务逻辑
- 工具层提供通用功能
- 数据持久化存储
