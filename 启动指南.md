# 微信小程序启动指南

## 前置准备

### 1. 安装微信开发者工具

- 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 下载并安装最新版本的微信开发者工具
- 使用微信扫码登录

## 启动步骤

### 方法一：导入项目（推荐）

1. **打开微信开发者工具**
   - 启动微信开发者工具
   - 点击左上角"项目" → "导入项目"

2. **选择项目目录**
   - 项目目录：选择 `miniprogram` 文件夹所在的路径
   - 例如：`D:\Github\wen\miniprogram`

3. **配置 AppID**
   - **有 AppID**：在 `project.config.json` 中修改 `appid` 字段为你的小程序 AppID
   - **没有 AppID**：选择"测试号"或"不使用云服务"
     - 测试号可以正常开发，但无法发布
     - 如需发布，需要到微信公众平台注册小程序获取 AppID

4. **点击"导入"**
   - 等待项目加载完成

5. **编译运行**
   - 点击工具栏的"编译"按钮（或按快捷键 Ctrl+B）
   - 在模拟器中查看效果

### 方法二：新建项目后替换文件

1. **创建新项目**
   - 打开微信开发者工具
   - 点击"+"创建新项目
   - 选择"小程序"
   - 填写项目名称，选择项目目录
   - AppID 选择"测试号"

2. **替换文件**
   - 将本项目的所有文件复制到新创建的项目目录中
   - 覆盖同名文件

3. **编译运行**
   - 点击"编译"按钮

## 常见问题

### Q1: 提示"未找到 app.json"？
**A:** 确保项目目录选择的是 `miniprogram` 文件夹，而不是外层文件夹。

### Q2: 编译报错"找不到模块"？
**A:** 
- 检查文件路径是否正确
- 确保所有 `.js` 文件都存在
- 检查 `import` 语句中的路径是否正确

### Q3: 如何获取 AppID？
**A:** 
1. 访问 https://mp.weixin.qq.com/
2. 注册/登录微信公众平台
3. 创建小程序
4. 在"开发" → "开发管理" → "开发设置"中查看 AppID

### Q4: 可以使用测试号开发吗？
**A:** 可以！测试号可以正常开发调试，只是无法发布到线上。开发阶段完全够用。

### Q5: 数据存储在哪里？
**A:** 所有数据存储在微信小程序的本地存储中：
- 在开发者工具中：点击"存储"标签页可以查看
- 在真机上：存储在微信小程序的本地存储空间

## 开发调试

### 查看控制台
- 点击工具栏的"调试器"按钮
- 在 Console 标签页查看日志输出

### 查看存储数据
- 点击工具栏的"存储"按钮
- 可以看到所有本地存储的数据

### 真机预览
1. 点击工具栏的"预览"按钮
2. 用微信扫码
3. 在手机上查看效果

### 上传代码
1. 点击工具栏的"上传"按钮
2. 填写版本号和项目备注
3. 上传后可在微信公众平台提交审核

## 项目结构说明

```
miniprogram/              ← 这是项目根目录
├── app.js                ← 应用入口
├── app.json              ← 全局配置
├── app.wxss              ← 全局样式
├── pages/                ← 页面目录
│   └── index/           ← 首页
├── services/             ← 服务层（业务逻辑）
├── utils/                ← 工具函数
└── project.config.json   ← 项目配置（包含 AppID）
```
